This is the gravitational kernel used in Seeßelberg (1996)[^1] and Bott (1998)[^2]. Because Beard (1976) is used for terminal velocity, `ecoll_table.csv` is a collision-efficiency table. Rows represent the radius of the larger droplet [µm], and columns represent the droplet radius ratio. The table values were adopted from Shima et al. (2020)[^3].
